# Kokomo Sunset Resort

This is a premium Next.js project bootstrapped with `create-next-app` for Kokomo Sunset Resort.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
